# Выход: Инсайт-карточки

**Когда:** цель — вытащить кросс-интервью инсайты. Дефолтный выход синтеза (S5–S7).
**Вход:** наггеты с кластерами → `score_insights.py`. Пиши карточки только на `insight`/`watchlist`.

## Формат карточки
```
### [ID] Заголовок — одно предложение
Формулировка: [наблюдение + интерпретация + следствие — не пересказ]
Держится для: [роли/сегменты] · НЕ для: [где не работает]
Распространённость: X из N интервью (роли: …) · Критичность: высокая/средняя/низкая
Тип: КОНСЕНСУС | НАПРЯЖЕНИЕ ([роль A] vs [роль B] — в чём)
Доказательства (только verified + support=yes):
  - [Интервью, роль] «дословная цитата» (Lxx)
  - [Интервью, роль] «…» (Lxx)
Контр-доказательства: [кто не подтвердил / что противоречит / где тишина]
Уверенность: высокая/средняя/низкая (по триангуляции + заземлённости)
Следствие / рекомендация: [что делать; для кого]
```

## Правила
- Паттерн = ≥k разных интервью с verified-цитатой (не k цитат от одного).
- Каждая карточка проходит контр-доказательства.
- Скоуп по роли: «для фондовиков», не «для сотрудников».
- Сортировка: `insight` по частота×критичность → `watchlist` (напряжения, редкое-критичное). `weak` — в приложение.
- Опционально → HTML-борд: `render_board.py provenance.json`.
