# Выход: Decision-memo (меморандум для решения)

**Когда:** цель — свести интервью в короткий документ под конкретное решение стейкхолдеров.
**Вход:** инсайты + (если есть) сравнение человек↔ИИ. Короче инсайт-карточек, заточен под 1 решение.

## Формат (1–2 страницы)
```
# Решение: [вопрос, на который отвечаем]
## Рекомендация
[одно чёткое действие/выбор + для кого]
## На чём основано (3–5 инсайтов)
- [инсайт] — X из N интервью, критичность — «цитата» (Интервью, Lxx)
## Риски и контраргументы
- [что против; кто не согласится; где данных мало]
## Чего мы НЕ знаем
- [открытые вопросы, пробелы, тишина в данных]
## Уверенность: высокая/средняя/низкая
[по триангуляции + заземлённости + n интервью]
```

## Правила
- Рекомендация одна и конкретная. Если данных на неё нет — так и напиши, не притягивай.
- Обязательный раздел контраргументов и «чего не знаем» — иначе это не решение, а презентация.
- Каждая опора — verified-цитата + число интервью. Латентные выводы (eNPS и т.п.) помечай как требующие человека.
- n<k → «пилот, не основание для необратимого решения».
