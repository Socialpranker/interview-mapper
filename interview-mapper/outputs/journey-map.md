# Выход: Journey Map (карта пути)

**Когда:** цель — путь посетителя/сотрудника/клиента по стадиям с эмоциями и трениями.
**Вход:** наггеты, размеченные по стадии пути (не по теме).

## Стадии (адаптируй под контекст)
- Посетитель: Узнал → Дошёл/вход → Экспозиция → Доп. услуги → Выход → После (отзыв/возврат).
- Сотрудник/процесс: Инициатива → Согласование → Исполнение → Сдача → Пост-анализ.
- Клиент: Осознание → Поиск → Выбор → Онбординг → Использование → Удержание.

## Формат (матрица стадия × измерение)
```
## Стадия: [название]
Действия: [что делает]
Эмоция: [+/−, интенсивность] — линия эмоций по стадиям
Боли/трения: [список] — «цитата» (Интервью, Lxx)
Точки восторга: — «цитата» (Lxx)
Кто затронут / прозрачность: [для сотрудника — узнаёт ли вовремя]
Распространённость: X из N интервью
```
В конце — **линия эмоций** (пик/провал по стадиям) и **top-3 точки трения** по частота×критичность.

## Правила
- Каждое трение/восторг — verified-цитата + интервью.
- Разные роли/сегменты могут иметь РАЗНЫЕ карты — не усредняй в одну, если пути расходятся.
- Стадия без данных — «не покрыто интервью» (не выдумывай).
