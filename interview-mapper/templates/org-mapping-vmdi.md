# Линза: Орг-картирование сотрудника — кодбук ВМДИ (6 блоков · 40 кодов)

**Когда:** интервью с сотрудником ВМДИ по гайду сотрудников. Это РЕАЛЬНЫЙ кодбук проекта (не реконструкция).
**Единица:** код. Каждый код заполняется как **цитата → «Что говорит» → «→ Рекомендация»**.
**Формат вывода на код:**
```
**С3.2 | Инцидент vs паттерн** [категория: Инцидент / Паттерн / Не упоминалось]
Категория: ПАТТЕРН
Что говорит: [интерпретация — что этот код открывает про систему]
_«дословная цитата» (Lxx)_
→ Рекомендация: [конкретное действие; если нет — «—»]
```
Типы кодов: **Открытый текст** (цитата+интерпретация), **Категория** (закрытый выбор + цитата), **Список** (набор из справочника + цитата). Если темы не было — «Не упоминалось».
**Δ (1–5)** — при сравнении ИИ↔Человек: 1 полное расхождение · 3 частичное · 5 полное совпадение. Считается по блокам (см. `references/rubric.md`).
**Правило системного приоритета (Блок 3):** совпадение эпизода у **≥3 сотрудников из разных отделов = системный приоритет** (в синтезе это триангуляция `score_insights --k 3` по ролям).

## БЛОК 1. Профессиональный путь и динамика отношений с музеем
_Цель: контекст присутствия, осознанность выбора, внутренний eNPS._
- **С1.1 Обстоятельства прихода** [открытый] — когда/откуда пришёл, что привело именно в этот музей; осознанный выбор или случайность.
- **С1.2 Альтернативы при трудоустройстве** [открытый] — рассматривал ли другие места; почему ВМДИ (тематика/люди/стабильность/иное).
- **С1.3 Разрыв ожидание/реальность** [открытый] — что оказалось неожиданным в первые месяцы (+/−).
- **С1.4 Мотиваторы и демотиваторы** [список+цитата] — Мотиваторы: Миссия/Коллеги/Профрост/Стабильность/Иное · Демотиваторы: Бюрократия/Зарплата/Управление/Иное.
- **С1.5 Внутренний eNPS** [категория+цитата] — Промоутер / Нейтрал / Детрактор / Не упоминалось. *(нестабильная ячейка → совет S3)*

## БЛОК 2. Опыт результативной профессиональной деятельности
_Цель: система в лучшем проявлении, воспроизводимость, инициатива снизу, межотдельное взаимодействие._
- **С2.1 Результативный кейс** [открытый] — проект/задача года, которым доволен; этапы.
- **С2.2 Источник инициативы** [категория] — Снизу / Сверху / Ситуативно / Не упоминалось.
- **С2.3 Межотдельное взаимодействие** [открытый] — кто вовлечён, как строилась коммуникация.
- **С2.4 Скорость решений и согласований** [открытый] — уровни согласований; задержки даже в лучшем кейсе = системный сигнал.
- **С2.5 Скрытые риски / хрупкость** [открытый] — что могло пойти не так, но не пошло.
- **С2.6 Культура признания** [категория+цитата] — Отмечено руководством / Коллегами / Посетителями / Не отмечено / Не упоминалось. *(нестабильная → совет)*
- **С2.7 Воспроизводимость результата** [категория] — Системный процесс / Личный энтузиазм / Смешанно / Не упоминалось.
- **С2.П Профильный вопрос** [открытый] — зависит от функции сотрудника; дословная цитата.

## БЛОК 3. Опыт операционных сбоев и системных ограничений
_Цель: карта системных сбоев. Каждый эпизод — точка, где ломается процесс._
- **С3.1 Эпизод сбоя** [открытый] — ситуация с несоразмерными затратами.
- **С3.2 Инцидент vs паттерн** [категория] — Инцидент / Паттерн / Не упоминалось.
- **С3.3 Атрибуция причины** [список] — Компетентность / Процесс / Ресурсы / Коммуникация / Смешанно / Не упоминалось.
- **С3.4 Зона ответственности** [открытый] — кто мог исправить, но не исправляет; проблема «ничья».
- **С3.5 Попытки исправления** [открытый] — предпринимались ли, результат, где «стена».
- **С3.6 Оценка потерь** [открытый] — временны́е/финансовые потери.
- **С3.7 Альтернативная стоимость** [открытый] — что делал бы с высвободившимися ресурсами.
- **С3.П Профильный вопрос** [открытый] — разрыв позиционирование/реальность по функции.

## БЛОК 4. Внутренние коммуникации и трансляция стратегии
_Цель: как стратегия доходит до сотрудников, инфоразрывы, силосы, инициатива снизу._
- **С4.1 Каналы получения информации** [открытый] — формальные и неформальные.
- **С4.2 Прозрачность управления** [открытый] — узнавал ли о решении постфактум.
- **С4.3 Проникновение стратегии** [категория+цитата] — Артикулирует / Частично / Не артикулирует / Не упоминалось.
- **С4.4 Межотдельные силосы** [список] — с кем эффективно, с кем затруднено; конкретные пары.
- **С4.5 Проходимость инициативы снизу** [категория+цитата] — Есть / Формально есть, не работает / Нет / Не упоминалось.
- **С4.6 Субъектность сотрудника** [открытый] — конкретные эпизоды влияния на решения.

## БЛОК 5. Посетительский опыт через призму профессионального наблюдения
_Цель: наблюдения сотрудников о посетителях — недооценённый источник; паттерны вне анкет._
- **С5.1 Запомнившийся эпизод** [открытый] — последнее взаимодействие с посетителем.
- **С5.2 Паттерн vs исключение** [категория] — Паттерн / Исключение / Не упоминалось.
- **С5.3 Повторяющиеся запросы / затруднения** [список] — точки трения на пути посетителя.
- **С5.4 Сегментация аудитории** [открытый] — различия аудитории по направлениям.
- **С5.5 Retention / постоянные посетители** [открытый] — причина возвращения; отслеживается ли.
- **С5.6 Типичный посетитель + барьеры** [открытый] — кто приходит; что мешает остальным (структурные vs операционные барьеры).
- **С5.7 Приоритет изменения в опыте** [открытый] — один аспект от входа до выхода.

## БЛОК 6. Приоритеты и вектор развития
